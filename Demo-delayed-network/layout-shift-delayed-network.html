Explanation

This demo shows how a delayed network request can cause layout shifts when an image is loaded.

In this example, the image is initially loaded with a base64 placeholder. When the actual image is loaded after a simulated network delay, it causes the content below the image to shift down.

The setTimeout function simulates a network delay, causing the image to load after a short delay.

This demonstrates the importance of reserving space for images to prevent layout shifts.
