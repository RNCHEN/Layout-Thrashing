This is some additional content that remains in place even when the image loads. It will be pushed down and shift once the delayed image gets loaded.

Explanation

This demo shows how reserving space for an image can prevent layout shifts caused by delayed network loading.

In this example, the width and height attributes of the <img> tag are used to reserve space for the image before it loads. This prevents the content below the image from shifting when the image is loaded.

The setTimeout function simulates a network delay, causing the image to load after a short delay.
